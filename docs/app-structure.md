# App Structure — FireOps HQ

## Main Navigation

1. Dashboard

2. Incidents

3. Time

4. Expenses

5. Crew

6. Fleet

---

## Screen Breakdown

### Dashboard

- Active incidents

- Quick stats (hours, expenses)

- Alerts (missing data, expiring items)

---

### Incidents

- List of incidents

- Create incident

- Incident detail:

  - assigned crew

  - assigned trucks

  - shifts

  - summary stats

---

### Time

- Log new entry

- View by incident

- View by crew member

---

### Expenses

- Add receipt

- View list

- Filter by category / incident

---

### Crew

- List of personnel

- Detail view

- Assignments

---

### Fleet

- List of trucks

- Status

- Notes

---

## Design Principles

- Keep screens simple

- Avoid deep navigation

- Prefer flat structure

- Optimize for speed and minimal taps
