# FireOps HQ

Mobile operations app for wildland firefighting contractors — track
incidents, time, expenses, crew, fleet, and shift tickets in the field.

- **Marketing site:** https://fireopshq.com
- **App (web):** https://app.fireopshq.com
- **iOS / Android:** packaged via Despia, see `docs/despia-setup.md`

## Tech

- React + Vite + TypeScript
- Tailwind CSS
- Lovable Cloud (Postgres + Auth + Storage + Edge Functions)

## Local development

```bash
npm install
npm run dev
```
