## Upload order (App Store Connect)

Upload in numbered order — Apple shows the first 3 in search results, so the
order is intentional: dashboard hook, then proof points (incidents, shift
tickets), then breadth (expenses, crew, fleet/offline).

## Other store sizes

App Store Connect will auto-scale 6.7" screenshots down for 6.5" and 5.5"
slots — you only need to upload one set unless you want device-specific copy.

For Google Play, these same images work for the **Phone screenshots** slot
(min 320px, max 3840px on either edge, 16:9 to 9:16 ratio).

## Regenerating

These were generated with the AI image tool at premium quality. To remake any
single one, regenerate at 1290×1920 then resize to 1290×2796 (cover-fit,
center crop). See chat history for prompts.

## Replacing with real device captures

Before submission, ideally retake at least screenshots 1–3 from a real
TestFlight build on a physical iPhone 15 Pro Max so reviewers see live UI.
Marketing-style frames are accepted, but real captures perform better in
conversion.
