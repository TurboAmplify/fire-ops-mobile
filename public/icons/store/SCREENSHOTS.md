## Upload order (App Store Connect)

Upload in numbered order — Apple shows the first 3 in search results, so the
order is intentional:

1. **Hero** — instant "what is this" via the dashboard
2. **Crew** — primary daily-use proof point
3. **Fleet** — operational breadth
4. **Offline** — the killer differentiator for field work
5. **Incidents** — real-time situational awareness
6. **Expenses** — paperwork-killer
7. **Shift Tickets** — OF-297 compliance, the contractor hook
8. **Audience** — explicit positioning for Contractors / VFDs / Agencies

Each screen communicates a **distinct** capability — no message is repeated.

## Other store sizes

App Store Connect will auto-scale 6.7" screenshots down for 6.5" and 5.5"
slots — you only need to upload one set unless you want device-specific copy.

For Google Play, these same images work for the **Phone screenshots** slot
(min 320px, max 3840px on either edge, 16:9 to 9:16 ratio).

## Regenerating

These were generated with the AI image tool at premium quality, then
center-cropped from 1290×1920 to the required 1290×2796 with PIL. See chat
history for prompts.

## Replacing with real device captures

Before submission, ideally retake at least screenshots 1, 2, and 5 from a
real TestFlight build on a physical iPhone 15 Pro Max so reviewers see live
UI. Marketing-style frames are accepted, but real captures perform better in
conversion.
